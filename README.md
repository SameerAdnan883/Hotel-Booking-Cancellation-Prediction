#  Hotel Booking Cancellation Prediction

An End-to-End Machine Learning Web Application that predicts whether a hotel booking will be cancelled based on customer booking details.

---

##  Project Overview

Hotel booking cancellations cause revenue loss and planning issues for hotels.

This project predicts whether a booking is likely to be cancelled using Machine Learning.

---

## ▶ Run Locally

Clone the repository

```bash
git clone <repository_link>
```

Move inside the project

```bash
cd Hotel-Booking-Prediction
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run Streamlit

```bash
streamlit run app.py
```
